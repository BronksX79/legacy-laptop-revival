# ChromeOS Flex Revival

## 🎯 Project Goal

Restore an ageing laptop to a fast, reliable machine suitable for everyday use and university work by combining affordable hardware upgrades with careful operating system evaluation.

---

## 🧭 Background

While helping my stepson move house, he came across his old HP 15-bw0xx laptop. Knowing I enjoy repairing and experimenting with computers, he offered it to me.

The laptop was running Windows 10 and had:

* AMD A6-9220 Dual-Core Processor
* 4 GB DDR4 RAM
* 1 TB mechanical hard drive
* A forgotten Windows password

He also warned me that it had become extremely slow before it was put away.

Rather than attempting to recover the existing installation, I decided to treat it as a restoration project.

---

## 🔍 Initial Assessment

The laptop had mainly been used for streaming sports and films from questionable websites.

Because I couldn't be confident the existing installation was free from malware, I chose **not** to connect it to my home network.

Instead, I performed a completely fresh installation of Windows 11 using an installation USB drive I already had.

Although Windows installed successfully, performance remained extremely poor. The system was slow to boot, applications took a long time to open, and overall usability was disappointing.

At that point, it became clear that reinstalling Windows alone was not going to solve the problem.

---

## 🔬 Research

Rather than replacing the laptop, I began researching ways to extend its useful life.

During this stage, I discovered **Ventoy**, a tool that allows multiple operating systems to be stored on a single bootable USB drive without repeatedly recreating installation media. This made testing different operating systems much easier.

I also compared recommendations from AI assistants and online communities.

The operating systems most frequently recommended were:

* Linux Mint XFCE
* MX Linux XFCE
* Zorin OS
* elementary OS
* Android-x86
* ChromeOS Flex

---

## 🔧 Hardware Upgrades

While researching components for my AI workstation project, I noticed that second-hand laptop upgrades were inexpensive.

To improve the laptop without spending much money, I installed:

* An additional 4 GB DDR4 SODIMM memory module (increasing total memory to 8 GB)
* A 120 GB SSD for operating system testing

The original 1 TB hard drive was removed, securely erased and retained for future projects.

**Total upgrade cost:** approximately **£9**.

---

## 💻 Operating System Evaluation

I tested several lightweight operating systems.

Linux distributions performed significantly better than Windows, but each had strengths and weaknesses.

Linux Mint XFCE and MX Linux were both lightweight and responsive.

Zorin OS was visually the most polished and became my personal favourite. However, I felt the free edition lacked some features, and introducing Linux to my daughter—who would eventually use the laptop for university work—would create an unnecessary learning curve.

I wanted a system that was:

* Simple
* Secure
* Lightweight
* Easy to maintain

Those requirements ultimately led me to ChromeOS Flex.

---

## ⚠️ Unexpected Problem

The first attempt to install ChromeOS Flex failed.

Although the installer started correctly, it could not detect the laptop's wireless network adapter. Because ChromeOS Flex requires an internet connection during setup, I was unable to continue with the installation.

Interestingly, the same wireless adapter worked perfectly under Linux, confirming that the issue was one of hardware compatibility rather than faulty hardware.

Instead of abandoning ChromeOS Flex, I researched compatible replacement wireless adapters.

---

## 🔄 Hardware Adaptation

While searching for a solution, I remembered an old Fujitsu laptop stored away.

Its wireless adapter was compatible with ChromeOS Flex, so I transplanted it into the HP laptop.

This introduced another unexpected challenge.

The original HP wireless card used a single antenna connection, while the Fujitsu replacement required two.

After further research, I confirmed that the replacement card would function correctly using only the primary antenna connection.

Once the laptop was reassembled, I attempted the ChromeOS Flex installation again.

This time the installation completed successfully, and the laptop booted with fully functional wireless networking.

---

## ✅ Outcome

The project transformed an almost unusable Windows laptop into a fast, responsive Chromebook-style device suitable for web browsing, productivity and university coursework.

For less than **£10** in upgrades, the laptop gained:

* 8 GB RAM
* SSD storage
* ChromeOS Flex
* Fully functional wireless networking
* A second useful life

The original 1 TB hard drive was retained for reuse in a future home server project.

---

## 🧠 Lessons Learned

* Older hardware should not be written off without investigation.
* An SSD upgrade provides one of the biggest real-world performance improvements.
* AI can dramatically speed up research, but practical testing remains essential.
* The best operating system depends on the intended user, not just technical performance.
* Hardware compatibility is often discovered through experimentation rather than specifications.
* Repurposing existing hardware is both economical and environmentally responsible.

---

## 🔮 Future Improvements

The restoration project is still ongoing.

Planned improvements include:

* 🔋 Replacing the original battery, which no longer holds a charge.
* 💾 Continuing to monitor long-term performance under ChromeOS Flex.

The laptop currently operates reliably when connected to mains power, so replacing the battery has been deferred while I prioritise other projects.
