ChromeOS Flex Revival
🎯 Project Goal

Restore an ageing laptop to a fast, reliable machine suitable for everyday use and university work by combining affordable hardware upgrades with careful operating system evaluation.

🧭 Background

While helping my stepson move house, he found his old HP 15-bw0xx laptop. Knowing I enjoy repairing and experimenting with computers, he gave it to me.

The laptop had:

Windows 10
4 GB DDR4 RAM
a 1 TB mechanical hard drive
a forgotten Windows password

He also warned me that it had become extremely slow before it was put away.

Rather than attempting to recover the existing installation, I decided to treat it as a restoration project.

🔍 Initial Assessment

The laptop had mainly been used for streaming sports and films from questionable websites.

Because I couldn't be confident the existing installation was clean, I chose not to connect it to my home network.

Instead, I performed a completely fresh installation of Windows 11 using an installation USB I already had.

Although Windows installed successfully, performance remained extremely poor.

At that point it became clear that the operating system itself wasn't the real solution.

🔬 Research

Instead of replacing the laptop, I researched ways to extend its useful life.

During this stage I discovered Ventoy, allowing multiple operating systems to be carried on a single USB drive without repeatedly recreating installation media.

I also compared recommendations from AI assistants and online communities.

The most commonly recommended operating systems were:

Linux Mint XFCE
MX Linux XFCE
Zorin OS
elementary OS
Android-x86
ChromeOS Flex
🔧 Hardware Upgrades

While researching my AI workstation project, I noticed affordable second-hand laptop components were available.

To improve the laptop without spending much money I installed:

Additional 4 GB DDR4 SODIMM RAM (bringing the total to 8 GB)
120 GB SSD for testing operating systems

The original 1 TB hard drive was removed, cleaned and kept for future projects.

Upgrade cost: approximately £9.

💻 Operating System Evaluation

I tested several lightweight operating systems.

Linux distributions performed significantly better than Windows, but each had drawbacks for the intended user.

Linux Mint XFCE and MX Linux were lightweight and responsive.

Zorin OS was visually the most polished and became my personal favourite, but I felt the free edition lacked some features and the overall Linux experience would introduce an unnecessary learning curve for my daughter, who would be using the laptop for university work.

I wanted something that was:

simple
secure
lightweight
easy to maintain

That ultimately led me to ChromeOS Flex.

⚠️ Unexpected Problem

The first attempt to install ChromeOS Flex failed.

Although the installer launched successfully, the operating system could not detect the laptop's wireless adapter. ChromeOS Flex requires a working network connection during setup, so I couldn't progress beyond the installation process.

Interestingly, the same wireless adapter worked correctly under Linux, confirming that the issue was one of hardware compatibility rather than a faulty component.

Rather than abandoning ChromeOS Flex, I began researching compatible replacement hardware.

🔄 Hardware Adaptation

I recovered an old Toshiba laptop from storage and discovered that its wireless adapter was compatible with ChromeOS Flex.

After transplanting the card into the HP laptop, I encountered another unexpected challenge.

The original HP adapter used a single antenna connection, while the replacement card required two antenna connectors.

After further research, I confirmed that the replacement card would operate correctly with only the primary antenna connected.

Once the laptop was reassembled, I attempted the ChromeOS Flex installation again.

This time the installation completed successfully and the laptop booted with fully functional wireless networking.

✅ Outcome

The project transformed an almost unusable Windows laptop into a fast, responsive Chromebook-like device suitable for web browsing, productivity and university coursework.

For less than £10 in upgrades, the laptop gained:

8 GB RAM
SSD storage
ChromeOS Flex
Full wireless networking
A second useful life

The original 1 TB hard drive was retained for reuse in a future home server project.

🧠 Lessons Learned
Don't assume old hardware needs replacing.
SSD upgrades provide one of the biggest performance improvements.
AI can speed up research, but practical testing remains essential.
Linux is excellent for experimentation, but the "best" operating system depends on the user.
Hardware compatibility is often discovered through testing rather than specifications.
Repurposing hardware can be both economical and environmentally responsible.

🔮 Future Improvements

The restoration project is still ongoing.

Planned upgrades include:

🔋 Replacing the original battery, which no longer holds a charge.
💾 Continuing to monitor long-term performance under ChromeOS Flex.

The laptop currently operates reliably when connected to mains power, so replacing the battery has been deferred while I prioritise other projects.
